# iPython


### Install

1. Open up virtualenv

    ```
    > workon <name>
    ```

2. Install ipython

    ```
    > pip install ipython
    ```


### Open a Notebook

```
> workon <name>
> ipython notebook
```
After you hit enter on ipython notebook, it will open up a Chrome tab. Your notebook will be empty the first time.
